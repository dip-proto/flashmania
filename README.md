# Flashmania - Testing Gemini-3.5-flash

Testing Gemini-3.5-flash.

2 prompts only, in distinct sessions, using [swival.dev](https://github.com/swival/swival):

- "In a markdown file, accurately describe the game Trackmania to someone who doesn't know the game at all, and is blind."
- "/goal Implement this in TypeScript."

Full agent traces are [available on Hugging Face](https://huggingface.co/datasets/jedisct1/agent-traces-flashmania).
